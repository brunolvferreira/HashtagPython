Gráfico de barras empilhadas para mostrar a distribuição de diferentes tipos de vendas ao longo do tempo¶

Considere que voc tenha uma empresa e queira mostrar a distribuio de diferentes tipos de vendas ao longo do tempo. Crie um grfico de barras empilhadas:

In [ ]:
import matplotlib.pyplot as plt

anos = [2018, 2019, 2020, 2021]
vendas_online = [5, 10, 15, 20]
vendas_loja = [10, 8, 6, 4]

plt.bar(anos, vendas_online, label='Online')
plt.bar(anos, vendas_loja, bottom=vendas_online, label='Loja')
plt.xlabel('Ano')
plt.ylabel('Vendas (em milhes)')
plt.title('Vendas Online vs Loja')
plt.legend()
plt.show()

import matplotlib.pyplot as plt
from cycler import cycler

anos = [2018, 2019, 2020, 2021]
vendas_online = [5, 10, 15, 20]
vendas_loja = [10, 8, 6, 4]

# rc = run configuration (configurao quando executa)
plt.rc('axes', prop_cycle=(cycler('color', plt.get_cmap('Pastel2').colors)))
 
plt.bar(anos, vendas_online, label='Online')
plt.bar(anos, vendas_loja, bottom=vendas_online, label='Loja')

# forar os anos a serem exibidos como inteiros
plt.xticks(ticks=anos)

plt.xlabel('Ano')
plt.ylabel('Vendas (em milhes)')
plt.title('Vendas Online vs Loja')
plt.legend()
plt.show()
