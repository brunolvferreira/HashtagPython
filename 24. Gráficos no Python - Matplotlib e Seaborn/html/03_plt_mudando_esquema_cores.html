Gráfico de pizza para mostrar a distribuição de categorias de produtos em uma loja¶

Suponha que voc tenha uma loja e queira mostrar a distribuio de categorias de produtos em sua loja. Voc pode usar um grfico de pizza para mostrar a distribuio de categorias de produtos em sua loja:

In [ ]:
import matplotlib.pyplot as plt

categorias = ['Eletrnicos', 'Roupas', 'Alimentos', 'Livros']
quantidades = [45, 30, 15, 10]  # valores em porcentagem

plt.pie(quantidades)
plt.title('Distribuio de Categorias de Produtos em Porcentagem')
plt.show()

import matplotlib.pyplot as plt

categorias = ['Eletrnicos', 'Roupas', 'Alimentos', 'Livros']
quantidades = [45, 30, 15, 10]
destaque = [0.1, 0, 0, 0]  # destacar a primeira fatia

plt.pie(quantidades, labels=categorias, autopct='%.1f', explode=destaque)
plt.title('Distribuio de Categorias de Produtos em Porcentagem')
plt.show()

O autopct usado para mostrar os valores percentuais. O %1.1f%% usado para mostrar um dgito aps o ponto decimal.

Mudando o esquema de cores: https://matplotlib.org/stable/tutorials/colors/colormaps.html

In [ ]:
import matplotlib.pyplot as plt

categorias = ['Eletrnicos', 'Roupas', 'Alimentos', 'Livros']
quantidades = [45, 30, 15, 10]
destaque = [0.1, 0, 0, 0]  # destacar a primeira fatia

cores = plt.get_cmap('Pastel2').colors

plt.pie(quantidades, labels=categorias, colors=cores, autopct='%.1f', explode=destaque)
plt.title('Distribuio de Categorias de Produtos em Porcentagem')
plt.show()
